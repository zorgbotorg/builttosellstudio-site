# Built to Sell Studio mini-site

This is a single-page website intended for business verification (e.g., Revolut Pro) and traffic handoff to Etsy.

## Free hosting options

### Option A: GitHub Pages (free subdomain)
- Domain format: `https://<username>.github.io/<repo>/`
- Steps:
  1. Create a public GitHub repo (e.g., `builttosellstudio-site`)
  2. Upload `index.html`
  3. In repo settings → Pages → Deploy from branch `main` `/root`

### Option B: Netlify Drop (fastest)
- Domain format: `https://<random>.netlify.app`
- Drag and drop the folder at https://app.netlify.com/drop
